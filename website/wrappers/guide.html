Teodor Sigaev and Oleg Bartunov of the OpenFTS project at
http://openfts.sourceforge.net
have developed a perl interface to the Snowball stemmers. It is written using C and provides OO and plain interfaces. Dowload from the above tarball. Comments and queries should be directed to Teodor and Oleg.

There is a new Perl wrapper (2006) from Marvin Humphrey of Rectangular Research. See
http://search.cpan.org/search?query=lingua%3A%3Astem%3A%3Asnowball
http://search.cpan.org/search?query=lingua%3A%3Astopwords
